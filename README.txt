EXPRESS 2
=========

A minimal blogging engine that takes a set of entries stored in plain text
files and generates a set of static interlinked HTML files.


HOW TO USE & DOCUMENTATION

See http://david.hgbrg.se/blog/express2.html.


DEPENDENCIES

* Unix (Linux, OS X etc.)
* PHP 5.3+ (http://php.net)
* PECL HTTP extension (http://pecl.php.net/package/pecl_http)
* Python and rst2html.py from the docutils package (http://docutils.sourceforge.net/)
